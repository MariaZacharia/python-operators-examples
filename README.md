# Python Operators Examples
This repository contains Python examples demonstrating various types of operators

## Contents
`operators_examples.py` - Python script with 8 examples of operators:
1. **Assignment Operator w/ string**(`=`)
2. **Assignment Operator w/ integer & tuple** (`+=`)
3. **Assignment Operator w/list** (`=`)
4. **Relational (comparison) Operator** (`==`)
5. **Relational (comparison) Operator** (`>=`)
6. **Membership Operator** (`in`)
7. **Membership Operator** (`not in`)
8. **Arithmetic Operators** (`+`, `-`, `/`, `//`, `**`, `%`)
